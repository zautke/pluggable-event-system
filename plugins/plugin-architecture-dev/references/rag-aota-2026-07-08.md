# RAG AOTA 2026-07-08

This contract set was implemented against the following basis on July 8, 2026.

## Runtime Targets

- .NET target: `net10.0` with C# `14.0`.
- C# serialization: `System.Text.Json` source generation via `JsonSerializerContext`.
- JSON Schema exporter: `System.Text.Json.Schema.JsonSchemaExporter`, introduced before .NET 10 and available to .NET 10 consumers.
- Wire schema: JSON Schema draft 2020-12.

## Provider Targets

- Ollama base URL: `https://ollama2.braisenly.com`.
- Ollama LLM model: `deepseek-v4-flash:cloud`.
- Ollama embedding models: `nomic-embed-text-v2-moe:latest` and `bge-m3:567m`.
- OpenAI LLM model: `gpt-4o-mini`.
- Vector stores: Qdrant and ChromaDB.
